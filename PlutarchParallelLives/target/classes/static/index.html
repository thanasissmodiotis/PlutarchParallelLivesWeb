Supported Input Formats

Single Files: .tsv, .csv

Folders:

  • Project Import: Contains tem.tsv + gpm.tsv (pre-clustered)
  • Schema Evolution: Contains *SchemaHeartbeat.tsv, *tables_DetailedStats.tsv, *transitions.csv
  • Data Folder: Any folder with .tsv or .csv files

Quick Start

  • Load data using "Load File" or "Load Folder"
  • Configure and apply clustering parameters
  • Click "Show Diagram" to visualize
  • Use dropdowns to adjust view settings

Diagram Colors

  • Light Blue (first column): Entity is alive
  • Light Red (first column): Entity is dead
  • Blue: Entity birth
  • Green shades: Active with measurements
  • Black: Entity death
  • White/Gray: Inactive period

Patterns

  • Multiple Births (blue): Many entities born simultaneously
  • Multiple Updates (orange): High concurrent update activity
  • Multiple Deaths (red): Many entities dying together
  • Ladder (magenta): Sequential lifecycle pattern

Keyboard Shortcuts

  • Ctrl + S: Save data
  • Ctrl + E: Export project
  • Shift + ?: Open help
